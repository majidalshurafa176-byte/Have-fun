# Have Fun Addon Features

## Block Transformation
The block transformation feature allows players to change blocks into different types, enhancing their creativity and gameplay experience. This feature is designed to promote exploration and experimentation within the game world.

## Supernatural Entities
This addon introduces various supernatural entities that players can encounter. These entities offer unique interactions and challenges, enriching the gaming experience.

## Darkness Effects
Players can experience darkness effects that add ambiance and challenges to the game, creating a more immersive atmosphere. This feature alters visibility and adds mystery to the gameplay.

## Installation Instructions
1. Download the addon from the official source.
2. Extract the files to your game directory.
3. Enable the addon in your game settings.
4. Restart the game to apply the changes.

Enjoy your adventures with the Have Fun addon!
